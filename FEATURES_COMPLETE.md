# Features Reference

## Conversion Tools
- Word to PDF
- Excel to PDF
- PowerPoint to PDF
- Images to PDF
- PDF to Images
- PDF to Word
- PDF to Excel
- PDF to PowerPoint
- Split PDF
- Merge PDF
- Compress PDF

## AI Summarizer
- PDF summarization with configurable length
- Asynchronous task processing
- Task status and result endpoints

## Platform Support
- Local Java runtime
- Docker container with LibreOffice

## Security
- Rate limiting
- CORS configuration
- Security headers (CSP, referrer policy, frame options)
